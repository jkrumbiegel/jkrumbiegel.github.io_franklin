I'm a neuroscience doctoral candidate at the Institute for Systemic Neuroscience in Hamburg, Germany, researching how prior expectations influence speech perception. For my work, I mostly use the Julia programming language and occasionally write about interesting aspects here.

I'm a main contributor to the Makie plotting ecosystem for which I wrote the layouting system MakieLayout. I also maintain a couple of Julia packages such as the data wrangling macro tool Chain.jl, the tweening library Animations.jl and the regex helper ReadableRegex.jl.
